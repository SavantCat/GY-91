# GY-91

---

##Description

* Chip: MPU-9250 + BMP280
* Power supply: 3-5v (internal low dropout regulator)
* Chip built 16bit AD converter, 16-bit data output
* Gyroscopes range: ± 250 500 1000 2000 ° / s,Acceleration range: ± 2 ± 4 ± 8 ± 16g
* 10 DOF
